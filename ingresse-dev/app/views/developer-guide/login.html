Login

Para desenvolver aplicativos que resgatam dados do usuário, é preciso que os usuários tenham dado permissão a sua aplicação para acessar os dados dele. Nesse caso, o usuário precisa se identificar e conceder essa permissão. Para isso, a Ingresse desenvolveu um mecanismo de identificação próprio, evitando que aplicativos de terceiros manipulem os dados de acesso a conta de nossos usuários.

O processo de autenticação envolve alguns elementos:

  • Token: a chave de acesso fornecida a você para poder criar aplicações credenciadas.
  • Url de retorno: resultado da função de timestamp que cria uma sessão válida de acesso com a API Ingresse.
