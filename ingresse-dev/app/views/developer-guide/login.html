Login

O sistema de login da Ingresse é uma forma rápida e segura de os usuário garantirem acesso a seus dados e compras.

O fluxo de login do usuário envolve 3 etapas: a solicitação de autorização pelo seu site ou aplicativo, a identificação do usuário em nossa tela de login ou criação de conta e o retorno para o seu site.

Solicitando o usertoken

Para solicitar o usertoken faça uma requisição usando o método /authorize. Essa requisição irá retornar a página HTML de login de usuário da Ingresse. Através dessa página o usuário irá informar seu email e senha autorizando que o respectivo aplicativo acesse seus dados da Ingresse.

Por padrão, o retorno desse login é um JSON com a seguinte estrutura:

{
    token: value,
    userId: value
}

Porém, na requisição do método /authorize você pode enviar também via GET o parâmetro returnurl com a URL de retorno no sucesso do login. Dessa forma, no sucesso do login a API irá redirecionar o resultado para a URL em returnurl enviando via GET as mesmas informações acima: token e userId. Exemplo:

Para retornar automaticamente para a url "http://www.meusite.com.br/meu-evento/pagamento", seria realizada a requisição: https://api.ingresse.com.br/authorize?returnurl=http%3A%2F%2Fwww.meusite.com.br%2Fmeu-evento%2Fpagamento
